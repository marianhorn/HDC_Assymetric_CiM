Crossover/mutation operator constants from hdc_infrastructure/asymItemMemory.c:
- Active pipeline for this test: PIPELINE_CUSTOM
- Custom crossover schedule constants:
  - ADAPTIVE_CHUNK_ALPHA=1.0
  - ADAPTIVE_CHUNK_REL_WIDTH=0.2
  - K = GA_MAX_FLIPS_CIM (events per feature block) = 2048
  - G = GA_DEFAULT_GENERATIONS = 128
  - mu_min = max(2, round(0.005 * K))
  - mu_max = max(mu_min + 1, round(0.15 * K))
  - t = g / (G - 1) for G>1, else 1
  - mu(g) = mu_min * ((mu_max / mu_min) ^ (t ^ alpha))
  - chunk size sampled uniformly from:
    - c_min = max(1, floor(mu(g) * (1 - 0.2)))
    - c_max = max(c_min, floor(mu(g) * (1 + 0.2)))
  - sample chunk sizes until all K events are consumed (last chunk clamped to remaining)
  - random offset before chunking: offset ~ Uniform{0..K-1}
  - per chunk parent source: 50/50 between parent A/B
  - per feature-block crossover gate: if rand < crossover_rate do chunk-mixing, else copy parent A block
- Custom mutation schedule constants:
  - ADAPTIVE_MUTATION_BETA=0.8
  - L = transitions = num_levels - 1 = 99
  - max_step(g) = round(L * (1/L)^(t^beta)), clamped to [1, L]
  - per event mutation gate: if rand < mutation_rate
  - step magnitude sampled uniformly in [1, max_step(g)]
  - step sign: +/- with 50/50 chance
  - periodic wrap in domain 1..L (modulo-based)
- Custom invariants/fail-fast checks:
  - each feature block must have exactly GA_MAX_FLIPS_CIM events (sum(B_block)=K)
  - mismatch/out-of-range events cause hard error and exit
- Feature blocking behavior:
  - PRECOMPUTED_ITEM_MEMORY=1 => crossover/mutation done independently per feature
  - number of feature blocks = num_features = 32

Naive pipeline operators (for reference; not active in this test):
- Crossover gate per child: rand < crossover_rate
- If crossover: each gene chosen 50/50 from parent A/B
- Else: child is parent A copy
- Mutation loop runs gene_count times; each step mutates with probability mutation_rate
- Mutation transfers exactly 1 count from a donor gene (>0) to a receiver gene
- Donor search max tries = 2 * gene_count
