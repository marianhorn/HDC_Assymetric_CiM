Detailed Description

Implements functions for generating and managing item memory used in hyperdimensional computing.

This file provides functionality to initialize, manage, and manipulate item memory vectors. Item memory is a key component in HDC and stores base vectors for encoding input data. The implementation supports both bipolar and binary data representations.

Functions in this file include initialization of item memory for discrete and continuous items, vector interpolation, storing/loading item memory to/from files, and generating orthogonal vectors.

void load_item_mem_from_bin ( struct item_memory *  item_mem,
const char *  filepath,
int  num_items 
)

Loads item memory vectors from a binary file.

This function reads vectors from a binary file and initializes the item memory structure.

The binary file must have the following layout:

  • Each vector is stored sequentially.
  • Each vector consists of VECTOR_DIMENSION elements.
  • The data type of each element must match the expected type:
    • int for bipolar mode (values: -1 or 1).
    • bool for binary mode (values: 0 or 1).

When reading the file, the function:

  • Initializes the item memory structure with num_items vectors.
  • Reads num_items * VECTOR_DIMENSION elements from the binary file and assigns them to the vectors.
Parameters
item_memA pointer to the item memory structure to be loaded.
filepathThe path to the binary file containing the item memory vectors.
num_itemsThe number of vectors to load into the item memory.
Note
Ensure that the binary file corresponds to the correct VECTOR_DIMENSION and num_items.

const char *  filepath 
)

Stores item memory vectors to a binary file.

This function writes the data of all vectors stored in the item memory to a binary file.

The layout in the binary file is as follows:

  • Each vector is stored sequentially.
  • Each vector consists of VECTOR_DIMENSION elements.
  • The data type of each element is vector_element, which is defined as:
    • int for bipolar mode (values: -1 or 1).
    • bool for binary mode (values: 0 or 1).

The binary file will contain num_vectors * VECTOR_DIMENSION elements, written as a contiguous array.

Parameters
item_memA pointer to the item memory structure.
filepathThe path to the binary file where the vectors should be stored.
Note
Ensure that the correct VECTOR_DIMENSION is used when reading this file.
