Detailed Description

Contains functions for preprocessing data before hyperdimensional encoding.

This file includes utilities to preprocess input datasets, such as:

  • Downsampling data to reduce its size while preserving structure.

The preprocessing steps are crucial for optimizing computational efficiency and ensuring compatibility with the hyperdimensional computing pipeline. Downsampling reduces the data size while maintaining temporal resolution, based on the DOWNSAMPLE factor.

Downsamples the input data by a specified factor.

This function reduces the size of the input dataset by selecting every DOWNSAMPLE-th sample. It adjusts both the data and their corresponding labels accordingly.

Parameters
dataThe original 2D array of input data with dimensions [original_size][NUM_FEATURES].
labelsThe array of labels corresponding to the input data, with original_size elements.
original_sizeThe total number of samples in the input data.
downsampled_dataPointer to the output 2D array for the downsampled data.
downsampled_labelsPointer to the output array for the downsampled labels.
newSizePointer to a variable to store the new size of the downsampled data.
Note
