\doxysubsection{Detailed Description}
Implementation of functions for managing and debugging hyperdimensional vectors. 

This file contains functions to initialize, free, and debug vectors used in hyperdimensional computing (HDC). Vectors are represented as arrays of elements and can be either binary or bipolar, depending on the configuration.


\begin{DoxyItemize}
\item {\bfseries{\mbox{\hyperlink{structVector}{Vector}} Initialization\+:}} Vectors are allocated dynamically, with values initialized to default values based on the chosen mode\+:
\begin{DoxyItemize}
\item Binary Mode\+: Elements are initialized to {\ttfamily false} (0).
\item Bipolar Mode\+: Elements are initialized to {\ttfamily -\/1}.
\end{DoxyItemize}
\item {\bfseries{\mbox{\hyperlink{structVector}{Vector}} Debugging\+:}} The file includes functions to print vector values, aiding in debugging.
\end{DoxyItemize}

\begin{DoxyNote}{Note}
Ensure proper memory management to avoid memory leaks. 
\end{DoxyNote}
