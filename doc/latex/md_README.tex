\hypertarget{md_README_autotoc_md2}{}\doxysection{Description}\label{md_README_autotoc_md2}
This project implements a well-\/structured framework for Hyperdimensional Computing (HDC).

{\bfseries{Evaluation Example\+:}} EMG-\/\+Classification of foot movements for the NOE-\/\+EMY project.

{\bfseries{Documentation\+:}}
\begin{DoxyItemize}
\item System and function details are available in the \href{./doc/html/mainpage_8h_source.html}{\texttt{ {\ttfamily /doc}}} folder and \href{./Abschlussbericht.pdf}{\texttt{ {\ttfamily Abschlussbericht.\+pdf}}}.
\item The development repository with all intermediate artifacts is available at\+: \href{https://gitlab.cs.fau.de/ew14ozom/emghandgestureshdc}{\texttt{ Development Repository}}.
\end{DoxyItemize}

\hypertarget{md_README_autotoc_md9}{}\doxysubsection{Customization Options}\label{md_README_autotoc_md9}
\hypertarget{md_README_autotoc_md10}{}\doxysubsubsection{Set System Parameters}\label{md_README_autotoc_md10}
Set system parameters in \href{./customModel/configCustom.h}{\texttt{ {\ttfamily config\+Custom.\+h}}}.\hypertarget{md_README_autotoc_md11}{}\doxysubsubsection{Implement a Data Reader}\label{md_README_autotoc_md11}
Implement a data-\/reading function specific to your problem and data in \href{./customModel/dataReaderCustom.c}{\texttt{ {\ttfamily data\+Reader\+Custom.\+c}}}.
\begin{DoxyItemize}
\item The training and testing data should be a 2D array of doubles, where each row represents a sample and each column represents a feature.
\item The training and testing labels should be a 1D array of integers, where each element corresponds to the label of a sample.
\end{DoxyItemize}\hypertarget{md_README_autotoc_md12}{}\doxysubsubsection{Modify the Model}\label{md_README_autotoc_md12}
The main file of the custom model is \href{./customModel/modelCustom.c}{\texttt{ {\ttfamily model\+Custom.\+c}}}.
\begin{DoxyItemize}
\item If working with categorical or discrete data, use the predefined functions in \href{./customModel/modelCustom.c}{\texttt{ {\ttfamily model\+Custom.\+c}}}.
\item If working on time-\/series classification, replace {\ttfamily \mbox{\hyperlink{trainer_8c_a4067122395f829b3344045208d94c086}{train\+\_\+model\+\_\+general\+\_\+data()}}} and {\ttfamily \mbox{\hyperlink{evaluator_8c_a8df1cdd88a7d44e3462237095794b178}{evaluate\+\_\+model\+\_\+general\+\_\+direct()}}} with {\ttfamily \mbox{\hyperlink{trainer_8c_a41885891fd0253febb456179bf221862}{train\+\_\+model\+\_\+timeseries()}}} and {\ttfamily \mbox{\hyperlink{evaluator_8c_aa90a74d814a24971672bd39182d314e6}{evaluate\+\_\+model\+\_\+timeseries\+\_\+direct()}}}.
\item For other types of data, extend the encoder to include additional encoding algorithms (e.\+g., for images, music, etc.). Use the following resources\+:
\begin{DoxyItemize}
\item \href{./hdc_infrastructure/vector.h}{\texttt{ {\ttfamily vector.\+h}}}\+: Provides vector structures (binary or bipolar, depending on \href{./customModel/configCustom.h}{\texttt{ {\ttfamily config\+Custom.\+h}}}) and related functions for initialization and modification.
\item \href{./hdc_infrastructure/operations.h}{\texttt{ {\ttfamily operations.\+h}}}\+: For binding, bundling, permutation, and similarity checks.
\end{DoxyItemize}
\end{DoxyItemize}\hypertarget{md_README_autotoc_md13}{}\doxysubsubsection{Other Features}\label{md_README_autotoc_md13}

\begin{DoxyItemize}
\item Include \href{./hdc_infrastructure/preprocessor.h}{\texttt{ {\ttfamily preprocessor.\+h}}} to downsample the data.
\item Include \href{./hdc_infrastructure/trainer.h}{\texttt{ {\ttfamily trainer.\+h}}} to process the training dataset, call the encoder on it, and set the associative memory.
\item Include \href{./hdc_infrastructure/evaluator.h}{\texttt{ {\ttfamily evaluator.\+h}}} to test the model using the testing dataset.
\item Include \href{./hdc_infrastructure/online_classifier.h}{\texttt{ {\ttfamily online\+\_\+classifier.\+h}}} to evaluate a trained model on live data.
\end{DoxyItemize}
