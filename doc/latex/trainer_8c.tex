\doxysubsection{Detailed Description}
Implementation of training functions for hyperdimensional computing (HDC) models. 

This file contains the implementation of functions for training HDC models with timeseries or general data. The training process involves encoding data into hypervectors and updating associative memory with class-\/specific bundled hypervectors.

Training is performed either using\+:
\begin{DoxyItemize}
\item Timeseries data, encoded with n-\/grams and handled as temporal sequences.
\item General data, where each sample is encoded independently.
\end{DoxyItemize}

The implementation supports both bipolar and binary modes\+:
\begin{DoxyItemize}
\item {\bfseries{Bipolar Mode\+:}} Incremental bundling of hypervectors is supported.
\item {\bfseries{Binary Mode\+:}} Hypervectors are pre-\/bundled for each class before updating associative memory due to the nature of majority voting.
\end{DoxyItemize}

The file includes mechanisms for handling stability checks on n-\/grams and ensures proper memory management for dynamically allocated hypervectors.
