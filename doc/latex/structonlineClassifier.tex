\doxysubsection{Detailed Description}
Represents the online classifier for real-\/time predictions. 

This structure handles batch-\/based online classification using a pre-\/trained associative memory and encoder.

Members\+:
\begin{DoxyItemize}
\item {\bfseries{assoc\+\_\+mem}}\+: Pointer to the associative memory used for classification.
\item {\bfseries{enc}}\+: Pointer to the encoder used for transforming input data into hypervectors.
\item {\bfseries{batch\+\_\+size}}\+: The number of samples to process in each batch. 
\end{DoxyItemize}
