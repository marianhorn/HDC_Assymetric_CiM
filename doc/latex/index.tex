\hypertarget{index_intro}{}\doxysection{Introduction}\label{index_intro}
This documentation covers the implementation of the Hyperdimensional Computing (HDC) framework for various use cases including EMG-\/based movement recognition and custom dataset evaluation. It is based on the work of \href{https://github.com/abbas-rahimi/HDC-EMG}{\texttt{ https\+://github.\+com/abbas-\/rahimi/\+HDC-\/\+EMG}} This is a reduced version of the project. For the complete version including intermediate results, debugging information and development artifacts refer to \href{https://gitlab.cs.fau.de/ew14ozom/emghandgestureshdc}{\texttt{ https\+://gitlab.\+cs.\+fau.\+de/ew14ozom/emghandgestureshdc}}

The framework consists of the following modules\+:
\begin{DoxyItemize}
\item {\bfseries{Associative Memory (\mbox{\hyperlink{assoc__mem_8c}{assoc\+\_\+mem.\+c}})}}\+: Implements storage and retrieval of hypervectors.
\item {\bfseries{Encoder (\mbox{\hyperlink{encoder_8c}{encoder.\+c}})}}\+: Encodes input data into hyperdimensional vectors.
\item {\bfseries{Item Memory (\mbox{\hyperlink{item__mem_8c}{item\+\_\+mem.\+c}})}}\+: Manages item memory for discrete and continuous signals.
\item {\bfseries{Operations (\mbox{\hyperlink{operations_8c}{operations.\+c}})}}\+: Contains operations such as binding, bundling, and permutation.
\item {\bfseries{Evaluator (\mbox{\hyperlink{evaluator_8c}{evaluator.\+c}})}}\+: Provides evaluation metrics and classification results.
\item {\bfseries{Trainer (\mbox{\hyperlink{trainer_8c}{trainer.\+c}})}}\+: Handles training of the associative memory.
\item {\bfseries{Preprocessor (\mbox{\hyperlink{preprocessor_8c}{preprocessor.\+c}})}}\+: Prepares and downsamples datasets.
\item {\bfseries{Main Files (e.\+g., \mbox{\hyperlink{modelCustom_8c}{model\+Custom.\+c}})}}\+: Entry points for specific use cases.
\end{DoxyItemize}\hypertarget{index_usage}{}\doxysection{Usage}\label{index_usage}
