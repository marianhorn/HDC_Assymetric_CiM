\hypertarget{index_intro}{}\doxysection{Introduction}\label{index_intro}
This documentation covers the implementation of the Hyperdimensional Computing (HDC) framework for various use cases including EMG-\/based movement recognition and custom dataset evaluation. It is based on the work of \href{https://github.com/abbas-rahimi/HDC-EMG}{\texttt{ https\+://github.\+com/abbas-\/rahimi/\+HDC-\/\+EMG}} This version of the project focuses on providing a clean and extensible framework for developing new HDC models. Non-\/essential components, intermediate results, and debugging artifacts have been omitted for clarity. For a comprehensive version of the project, including all development artifacts, refer to the \href{https://gitlab.cs.fau.de/ew14ozom/emghandgestureshdc}{\texttt{ FAU Git\+Lab Repository}}.

The framework consists of the following modules\+:
\begin{DoxyItemize}
\item {\bfseries{Item Memory (\mbox{\hyperlink{item__mem_8c}{item\+\_\+mem.\+c}})}}\+: Manages the discrete and continuous mappings for features and values into hyperdimensional space.
\item {\bfseries{Associative Memory (\mbox{\hyperlink{assoc__mem_8c}{assoc\+\_\+mem.\+c}})}}\+: Stores and retrieves class hypervectors efficiently.
\item {\bfseries{Encoder (\mbox{\hyperlink{encoder_8c}{encoder.\+c}})}}\+: Encodes input data into hyperdimensional vectors.
\item {\bfseries{Operations (\mbox{\hyperlink{operations_8c}{operations.\+c}})}}\+: Provides low-\/level vector operations, including binding, bundling, and similarity checks.
\item {\bfseries{Trainer (\mbox{\hyperlink{trainer_8c}{trainer.\+c}})}}\+: Handles training of the associative memory based on training dataset.
\item {\bfseries{Evaluator (\mbox{\hyperlink{evaluator_8c}{evaluator.\+c}})}}\+: Evaluates a trained model based on testing dataset.
\item {\bfseries{Online Classifier (\mbox{\hyperlink{online__classifier_8c}{online\+\_\+classifier.\+c}})}}\+: Evaluates a trained model based on batches of real time incoming test data.
\item {\bfseries{Preprocessor (\mbox{\hyperlink{preprocessor_8c}{preprocessor.\+c}})}}\+: Prepares and downsamples datasets.
\item {\bfseries{Main Files (e.\+g., \mbox{\hyperlink{modelCustom_8c}{model\+Custom.\+c}})}}\+: Entry points for specific use cases.
