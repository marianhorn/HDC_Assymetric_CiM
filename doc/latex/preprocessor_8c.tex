\doxysubsection{Detailed Description}
Contains functions for preprocessing data before hyperdimensional encoding. 

This file includes utilities to preprocess input datasets, such as\+:
\begin{DoxyItemize}
\item Downsampling data to reduce its size while preserving structure.
\end{DoxyItemize}

The preprocessing steps are crucial for optimizing computational efficiency and ensuring compatibility with the hyperdimensional computing pipeline. Downsampling reduces the data size while maintaining temporal resolution, based on the {\ttfamily DOWNSAMPLE} factor.
