\doxysubsection{Detailed Description}
Implements functionality for encoding EMG signals into hypervectors. 

The encoder maps raw EMG signals into high-\/dimensional representations (hypervectors) for further processing in Hyperdimensional Computing (HDC). It supports both spatial and temporal encoding. The encoded hypervectors are used for tasks such as classification and evaluation.

Key features include\+:
\begin{DoxyItemize}
\item Conversion of continuous EMG signals into discrete levels.
\item Encoding of individual timestamps (spatial encoding).
\item Aggregation of multiple timestamps into N-\/gram hypervectors (temporal encoding).
\item Compatibility with both bipolar and binary modes.
\end{DoxyItemize}

\begin{DoxyNote}{Note}
This file supports configurations for both precomputed and dynamically generated item memories through the {\ttfamily PRECOMPUTED\+\_\+\+ITEM\+\_\+\+MEMORY} macro.
\end{DoxyNote}
