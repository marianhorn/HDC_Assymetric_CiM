\doxysubsection{Detailed Description}
Represents the associative memory used for HDC. 

This structure holds the class vectors and their associated counts.

\item {\bfseries{num\+\_\+classes}}\+: The total number of classes in the associative memory.
\item {\bfseries{class\+\_\+vectors}}\+: Array of hypervectors, one for each class.
\item {\bfseries{counts}}\+: Array of integers tracking the number of samples per class.
