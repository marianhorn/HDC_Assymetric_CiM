\doxysubsection{Detailed Description}
Implements vector operations for hyperdimensional computing. 

This file contains functions for common operations on hypervectors, including\+:
\begin{DoxyItemize}
\item {\bfseries{Binding\+:}} Combines two vectors element-\/wise, either via multiplication (bipolar) or XOR (binary).
\item {\bfseries{Bundling\+:}} Aggregates multiple vectors, using summation (bipolar) or majority voting (binary).
\item {\bfseries{Permutation\+:}} Performs cyclic shifts on vectors for encoding temporal information.
\item {\bfseries{Similarity\+:}} Computes similarity metrics, such as cosine similarity (bipolar) or Hamming distance (binary).
\end{DoxyItemize}

The operations are optimized for both bipolar and binary vector modes, allowing flexibility in hyperdimensional computing applications. These functions form the core of data encoding, aggregation, and classification pipelines.
